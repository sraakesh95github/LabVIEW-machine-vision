JSON String to LVCLuster.vi

Owning Palette: Zen JSON File Handling

Requires: JKI JSON, VI Scripting

Platform(s): Windows

This VI serializes the input JSON string and opens a VI containing the LabVIEW datatype corresponding to the input JSON string.
